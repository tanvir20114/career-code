# Career Code
A job portal web application

## Live Site
https://career-code-3.web.app/

## Server
https://career-code-beta.vercel.app/

## Features
- Browse job listings
- Apply for jobs
- Employer job posting
- User authentication

## Technologies Used
### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Firebase Auth
